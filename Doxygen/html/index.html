PROP Grup 12.1
Compressió i descompressió de fitxers i imatges
Projecte de Programació

Es tracta de construir un entorn per a la compressió i descompressió de fitxers .txt i de imatges .ppm

Compressor i Descompressor

Fitxers .TXT

Algorismes

LZ78 : Lempel-Ziv, 1978

LZSS : Lempel–Ziv–Storer–Szymansk, 1982

LZW : Lempel–Ziv–Welch, 1984

Imatges .PPM

Algorisme

JPEG : Joint Photographic Experts Group, 1992


Com utilitzar l'instal·lador

Ens anem dins de la carpeta de src. Allà, obrim un terminal i executem "./compileAndRun.sh". Llavors, ens sortirà un menú desplegable amb les opcions disponibles, com aquest:

make compila el projecte.
run per executar el main.
-driver i el nom del driver que vols executar. Exemple: ./compileAndRun.sh -driver domini.algorithm.Driver__LZ78
-ctest Compila els tests.
-etest i el nom del test que vols executar. Exemple: ./compileAndRun.sh -etest domini.algorithm.LZWTest
-clean o -remove Borra els executables.
